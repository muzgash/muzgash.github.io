M.A.R.B.L.E.S.

MARBLES es del inglés Marvelous Arduino & Raspberrypi Biorreactor Lightweight Electronic System. Es un sistema electrónico controlado remotamente que ajusta la intensidad de la luz, prender y apagar la inyección de CO2 y medir la densidad de algas en un biorreactor pequeño.

Puede ser controlado desde un celular o vualquier navegador. Guarda un registro y las graficas de las medidas y la intensidad.

Toda la información (incluyendo esquemas electrónicos) puede encontrarse en el cuaderno de protocolo creado durante su desarrollo project MARBLES

Sistema de Seguridad con Raspberry Pi

Los sitemas o servicios de seguridad son muy costosos sobre todo para una casa. Un kit de RaspberryPi con cámara puede costar al rededor de $100 USD.

Desarrollé un sistema de seguridad que detecta personas y envía un reporte a través de un bot de telegram. Usando una RaspberryPi con cámara infrarroja puede obtener estas imágenes.

La documentación y los códigos del proyecto están aún por ser subidos a github, pero en el entretanto, estas son las herramientas que usé para su desarrollo:

  • RaspberryPi camera image capture
  • Pedestrian detection (right now updating the system to use YOLO algorithm)
  • Telegram bot python API
